# Awesome Periodic Table

An interactive periodic table built as a hobby project for my 6-year-old son Santiago.

I'm not a scientist — this was made with love and curiosity, not expertise. It may contain errors or oversimplifications. If you spot something wrong, feel free to open an issue.

## The spirit of it

This is what happens when a six-year-old's curiosity meets the power of AI. No frameworks, no design patterns, no engineering best practices — just a father trying to make science feel magical for his kid. The goal was never technical elegance. The goal was awesome.

## What it is

A visual, kid-friendly exploration of the periodic table with element details, body composition, and other fun science topics.
